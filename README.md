# Project Medicalapplication - Backend School

## Description du projet

Ce dépôt contient les services backend d'une application médicale microservices (services de configuration, découverte, passerelle, et services métier comme medecin, patient et rendez-vous).

## Fonctionnalités

- Gestion des médecins (CRUD)
- Gestion des patients (CRUD)
- Gestion des rendez-vous
- Service de configuration centralisé
- Service de découverte (Eureka / Consul)
- Passerelle API pour routage et sécurité
